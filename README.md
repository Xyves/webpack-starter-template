# Getting Started

## 1. Clone repo

```
$ git clone https://github.com/Xyves/webpack-starter-template.git 
```

## 2. Install packages

```
npm install
yarn install
```

## 3. Build

- Build the app:

```
$ npm run build
```

Or:

```
$ yarn run build
```

## 4. Server
- Run server:

```
$ npm run serve
```

Or:

```
$ yarn run serve
```

The server will be running on:

```
localhost:9000
```
